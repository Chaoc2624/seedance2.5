title: 快速开始
---

## 安装

要开始使用 YourAppName，你需要先安装依赖。

```bash
git clone https://github.com/your-org/your-app.git
cd your-app
npm install
```

## 配置环境

将 `.env.example` 复制为 `.env.local` 并填写必要的配置值：

```bash
cp .env.example .env.local
```

### 必需的环境变量

- `DATABASE_URL` — 数据库连接字符串（推荐使用 PostgreSQL）
- `AUTH_SECRET` — 用于认证会话的随机密钥
- `STRIPE_SECRET_KEY` — Stripe 支付密钥

## 启动开发服务器

使用以下命令启动开发服务器：

```bash
npm run dev
```

应用将在 `http://localhost:3000` 上可用。

## 项目结构

```
├── src/
│   ├── components/    # 可复用 UI 组件
│   ├── routes/        # TanStack Router 文件路由
│   ├── core/          # 核心业务逻辑
│   ├── config/        # 应用配置
│   └── contexts/      # React Context 提供者
├── content/
│   └── docs/          # 文档 MDX 文件
└── public/            # 静态资源
```
