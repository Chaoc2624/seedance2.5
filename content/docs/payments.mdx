## Overview

YourAppName integrates with Stripe for payment processing, supporting both one-time purchases and recurring subscriptions.

## Setting Up Stripe

1. Create a Stripe account at [stripe.com](https://stripe.com)
2. Get your API keys from the Stripe Dashboard
3. Set the environment variables:

```env
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
```

## Webhook Handling

Stripe webhooks are handled automatically via the `/api/payment/callback` endpoint. Supported events include:

- `checkout.session.completed` — Subscription created
- `invoice.payment_succeeded` — Recurring payment processed
- `customer.subscription.deleted` — Subscription cancelled
