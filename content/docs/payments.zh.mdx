## 概述

YourAppName 集成了 Stripe 支付系统，支持一次性购买和定期订阅。

## 配置 Stripe

1. 在 [stripe.com](https://stripe.com) 创建 Stripe 账号
2. 从 Stripe 控制面板获取 API 密钥
3. 设置环境变量：

```env
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
```

## Webhook 处理

Stripe Webhook 通过 `/api/payment/callback` 端点自动处理。支持的事件包括：

- `checkout.session.completed` — 订阅创建
- `invoice.payment_succeeded` — 定期支付处理
- `customer.subscription.deleted` — 订阅取消

## 测试支付

使用 Stripe 测试卡进行开发测试：

- **成功**: `4242 4242 4242 4242`
- **拒绝**: `4000 0000 0000 0002`
- **3D 验证**: `4000 0025 0000 3155`
